# 🏦 Bank Customer Churn Analysis Dashboard | Power BI

# 📌 Project Overview

This project presents an interactive **Bank Customer Churn Analysis Dashboard** built using **Power BI**. The dashboard helps financial institutions analyze customer behavior, identify churn patterns, monitor customer demographics, and evaluate key performance indicators that influence customer retention.

The dashboard enables business users to explore customer churn by geography, gender, balance, salary, and account activity through interactive visualizations.

---

# 🎯 Business Objectives

- Analyze customer churn across different countries.
- Monitor customer balances and estimated salaries.
- Compare churn rates by gender.
- Identify active and inactive customers.
- Support customer retention strategies using data-driven insights.

# 💡 Business Questions Answered

- Which country has the highest customer balance?
- Which country experiences the highest customer churn?
- Which gender has the highest churn rate?
- What percentage of customers remain active?
- How does salary differ between male and female customers?

---

# 📈 Key Insights

- Germany records the highest customer churn among the three countries.
- France has the highest total customer balance.
- Female customers account for a slightly higher proportion of churn.
- Active customers represent a significant portion of the customer base.
- Salary distribution is relatively balanced across genders.
